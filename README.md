1 - Clone o repositório:

# git clone https://github.com/alexsbcorrea/front-service-desk-agent-python.git

2 - Abra o repositório no Visual Studio Code;

3 - Instale as dependências do projeto

# npm install

4 - Execute o projeto

# npm run dev

5 - Se necessário, crie uma variável de ambiente com o link da API

NEXT_PUBLIC_BASE_URL=http://localhost:5000

5 - Acesse a página inicial

http://localhost:3000/
